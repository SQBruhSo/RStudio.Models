Configuración

Requisitos del Sistema

• Roblox Studio instalado

• Windows 10/11 (64-bit)

• 4GB RAM mínimo

• 2GB espacio libre

Configuración Recomendada

• Windows 11

• 8GB RAM o más

• Gráficos dedicados

• SSD para mejor rendimiento

Solución de Problemas

• Si no se abre: Verifica que Roblox Studio esté instalado

• Si es lento: Cierra otros programas

• Si hay error: Reinstala Roblox Studio
